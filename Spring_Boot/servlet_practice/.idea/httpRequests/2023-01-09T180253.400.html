HTTP 상태 400 – 잘못된 요청

타입 예외 보고

메시지 HTTP 헤더 행 [get http: ]이(가) RFC 7230을 준수하지 않아, 무시되었습니다.

설명 클라이언트 오류로서 인지된 어떤 문제로 인하여, 서버가 해당 요청을 처리할 수 없거나, 처리하지 않을 것입니다. (예: 잘못된 요청 문법, 유효하지 않은 요청 메시지 framing, 또는 신뢰할 수 없는 요청 라우팅).
